Inline Elements

So far, we've covered block elements (aka "flow content"). Another major type of content is the inline element, or "phrasing content". These are treated differently and can affect sections of text anywhere within a line.

Somtimes, you need to draw attention to a particular word or phrase. This can be done with the "em" tag, which emphasized the "sometimes".

Or, maybe you want to bold something. That's facilitated with the "strong" tag. Bold and italicized are possible like this, by using multiple nested inline elements.

Now, let's be clear: while "bold" is the action, it's not the correct term. The correct term is the term in the tag: strong. This is a difference between HTML and CSS. HTML will provide the semantic structure, while CSS will provide presentation information.
