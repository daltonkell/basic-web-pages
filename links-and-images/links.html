This particular page is about links! There are three types of links:

  • Absolute links, like to Mozilla Developer Network, which is a very good resource for web development stuff.

    It's possible to use absolute links to refer to pages in your own website, but hard-coding your domain everywhere can make for some tricky situations. It's usually a better idea to reserve absolute links for directing users to a different website.

  • Relative links, like to our extras page.

    Now, there are relative links called "root-relative links". Instead of being relative to the current
    directory, they are relative to the root of the project. They look like so:
    <a href='/'>home page</a>
    These aren't particularly applicable right now, but are super useful in terms of actual web pages when we have a real server.

Link Targets

Attributes alter the meaning of HTML elements, and sometimes you need to modify more than one
aspect of an element. For example, the <a> tag also accepts a "target" attribute, which defines
where to display the page when the user clicks a link. By defauly, most browsers replace the current page
with the new one. We can use the "target" attribute to as the browser to open a link in a new window/tab.
