The Cascade

CSS is applied to pages in a prioroty heirarchy, from lowest priority to highest priority:

  1. The browser’s default stylesheet
  2. User-defined stylesheets
  3. External stylesheets (that’s us)
  4. Page-specific styles (that’s also us)
  5. Inline styles (that could be us, but it never should be)
This means styles defined in subsequest steps override the previous ones. An example of this can be found at our dummy page; check out the code and the rendering!
